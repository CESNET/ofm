TSU Format to ns Convertor

ENTITY TSU_FORMAT_TO_NS IS

This component converts the Timestamp format of the TimeStamp Unit (the TS_NS output port) to pure nanosecond format. It does this by multiplying the upper 32 bits (that indicate the number of seconds) of the input by 10**9 and adding it to the lower 32 bits (that indicate the number of nanoseconds). Quartus and Vivado both manage to use DSPs for the multiplication, eventhough this description is behavioral. Input and output registers can be enabled.
