PCIe Addres Generator

ENTITY PCIE_ADDR_GEN IS

This module generates pcie headers acording to packets size. The module generates number of geneaders depending on INPUT_SIZE and BLOCK_SIZE. Number of generated headers is round up INPUT_SIZE/BLOCK_SIZE
